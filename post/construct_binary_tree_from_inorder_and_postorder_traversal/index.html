106. Construct_binary_tree_from_inorder_and_postorder_traversal

2020-04-24

Description:

Given inorder and postorder traversal of a tree, construct the binary tree.

Note: You may assume that duplicates do not exist in the tree.

Examples:

For example, given

inorder = [9,3,15,20,7] postorder = [9,15,7,20,3]

Return the following binary tree:

Thinking:

该题考察如何通过给定的中序遍历和后序遍历得到的列表，将该二叉树给重新序列化出来 考虑后序遍历中，最后一个节点一定是根节点，再通过该根节点在中序遍历列表中的位置，可以得到根节点对应的左子树和右子树 依次对每个左子树和右子树进行递归遍历，通过左右子树在后序列表中找到各子树的根节点，并再次通过根节点再在中序列表中的位置，找到对应的左右子树

class Solution(object):
    def buildTree(self, inorder, postorder):
        """
        :type inorder: List[int]
        :type postorder: List[int]
        :rtype: TreeNode
        """
        def helper(in_left, in_right):
          if in_left > in_right:
            return None
          val = postorder.pop()
          root = TreeNode(val)
          index = idx_map[val]

          root.right = helper(index+1, in_right)
          root.left = helper(in_left, index-1)
          return root

        idx_map = {val:index for index, val in enumerate(inorder)}
        return helper(0, len(inorder)-1)
