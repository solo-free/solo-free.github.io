19. Remove Nth Node From End of List

2020-03-22

Contents

Description:

Given a linked list, remove the n-th node from the end of list and return its head.

Examples:

Given linked list: 1->2->3->4->5, and n = 2. After removing the second node from the end, the linked list becomes 1->2->3->5.

Note:

Given n will always be valid.

Follow up:

Could you do this in one pass?

Thinking:

The normal way to remove a node from a list is finding the current_node’s prev node, and let the prev node.next = prev node.next.next In this question, we also use this way to remove node. Another thing we should do is finding the prev node from end. In this situation, we can use len(linked_list) - n (given node) to find the node, which should be removed.

Solution:

  • 一般解：

两次遍历 第一次遍历获取整个链表的长度， 第二次遍历，找到要被删除节点的前置节点，进行删除操作

    def removeNthFromEnd1(head, n):
        """
        :type head: ListNode
        :type n: int
        :rtype: ListNode
        """
        length = 0
        cur_node = head
        while cur_node:
          length += 1
          cur_node = cur_node.next
        cur_node = head
        index = length - n
        if index == 0:
          return cur_node.next
        for i in range(length):
          if i == index - 1:
            cur_node.next = cur_node.next.next
            return head
          else:
            cur_node = cur_node.next
  • 最优解：

two pointer 双指针法，将两个指针的间隔始终保持为 n, 当第二个指针遍历到链表的最后时，第一个指针即为要删除节点的前置节点

    def removeNthFromEnd1(head, n):
        """
        :type head: ListNode
        :type n: int
        :rtype: ListNode
        """
        dummy_node = ListNode(0)
        dummy_node.next = head
        l_node, r_node = dummy_node, dummy_node
        for i in range(n):
          r_node = r_node.next
        while r_node.next:
          l_node = l_node.next
          r_node = r_node.next
        l_node.next = l_node.next.next
        return dummy_node.next
