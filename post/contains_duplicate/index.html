Describtion:

Given an array of integers, find if the array contains any duplicates. Your function should return true if any value appears at least twice in the array, and it should return false if every element is distinct.

Example:

  • Example1: Input: [1,2,3,1] Output: true

  • Example2: Input: [1,2,3,4] Output: false

Thinking:

判断一个数组是否存在重复数值，可以通过比对去重后数组个数和原数组数组是否相等
也可通过遍历数组，通过一个set集合记录唯一数值，当出现遍历的值在 set集合中时，说明该值为重复出现的数值

    def containsDuplicate(nums):
    """
    :type nums: List[int]
    :rtype: bool
    """
    return len(nums) != len(set(nums))

    def containsDuplicate(nums):
set_result = set()
for num in nums:
  if num in set_result:
    return True
  set_result.add(num)
return False
