Description:

Implement an iterator over a binary search tree (BST). Your iterator will be initialized with the root node of a BST.

Calling next() will return the next smallest number in the BST.

 iterator.hasNext(); // return true
 iterator.next();    // return 15
 iterator.hasNext(); // return true
 iterator.next();    // return 20
 iterator.hasNext(); // return false

Note:

next() and hasNext() should run in average O(1) time and uses O(h) memory, where h is the height of the tree. You may assume that next() call will always be valid, that is, there will be at least a next smallest number in the BST when next() is called.

Thinking:

递归法：通过 BST的中序遍历，将遍历的结果按顺序存入 list 中, 初始化一个 index 变量，每次调用 Next()，HasNext()方法时，对 index 变量做累加

迭代法：自己构造一个 stack，用来存储中序遍历 bst 后的结果

 class BSTIterator(object):

    def __init__(self, root):
        """
        :type root: TreeNode
        """
        # initialize a index
        self.index = -1

        # initialize a ordered list
        self.inorder_ordered_list = []

        self._inorder(root)

    def _inorder(root):
      if not root:
        return

      self._inorder(root.left)
      self.inorder_ordered_list.append(root.val)
      self._inorder(root.right)

    def next(self):
        """
        @return the next smallest number
        :rtype: int
        """
        self.index = self.index + 1
        return self.inorder_ordered_list[self.index]

    def hasNext(self):
        """
        @return whether we have a next smallest number
        :rtype: bool
        """
        return self.index+1 < len(self.inorder_ordered_list)

 class BSTIterator(object):

    def __init__(self, root):
        """
        :type root: TreeNode
        """

        self.stack = []
        self._leftmost_inorder(root)

    def _leftmost_inorder(self, root)
        while root:
          self.stack.append(root)
          root = root.left

    def next(self):
        """
        @return the next smallest number
        :rtype: int
        """
        topmost_node = self.stack.pop()
        if topmost_node.right:
          self._leftmost_inorder(topmost_node.right)
        return topmost_node.val

    def hasNext(self):
        """
        @return whether we have a next smallest number
        :rtype: bool
        """
        return len(self.stack) > 0
