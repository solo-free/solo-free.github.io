Description:

Reverse a singly linked list.

Examples:

Input: 1->2->3->4->5->NULL
Output: 5->4->3->2->1->NULL

Note:

A linked list can be reversed either iteratively or recursively. Could you implement both?

Thinking:

如果将翻转 linked list 的重点放在 pointer 的指向翻转上，解决问题的思路会更简单些。 链表翻转可使用迭代的方法也可使用递归的方法。其关键点都是将当前节点(cur node)的 next 指向 上一个节点 (prev node)

    def reverseList(head):
    """
    :type head: ListNode
    :rtype: ListNode
    """
    prev = None
    cur_node = head
    while cur_node is not None:
      next_node = cur_node.next
      cur_node.next = prev
      prev = cur_node
      cur_node = next_node
    return prev

    def reverseList(head):
    """
    :type head: ListNode
    :rtype: ListNode
    """
    if head is None or head.next is None:
      return head
      prev = reverseList(head.next)
      head.next.next = head
      head.next = None
      return prev
