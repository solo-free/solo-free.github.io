344. Reverse_string

2019-12-07

Contents

      • Describtion:
      • Example:
      • Thinking:
      • Solution:

Describtion:

Write a function that reverses a string. The input string is given as an array of characters char[]. Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1) extra memory. You may assume all the characters consist of printable ascii characters.

Example:

  • Example 1: Input: [“h”,“e”,“l”,“l”,“o”]
    Output: [“o”,“l”,“l”,“e”,“h”]

  • Example 2: Input: [“H”,“a”,“n”,“n”,“a”,“h”]
    Output: [“h”,“a”,“n”,“n”,“a”,“H”]

Thinking:

翻转字符串，首先可以考虑使用递归的方式来解决，但递归在操作上相当于栈的调用，不符合题目中常量级的空间操作。 该题主要考察如何将数组中的字符串翻转, 可通过双指针法将遍历的元素收尾调换， 也可通过翻转数组实现元素 sort
注意该题需要 in_place 转换数组元素， 不可使用额外空间

    def reverseString(self, s):
    """
    Do not return anything, modify s in-place instead.
    """
    s[:] = s[::-1]

    def reverseString(self, s):
    """
    Do not return anything, modify s in-place instead.
    """
    left, right = 0, len(s)-1
    while left < right:
      s[left], s[right] = s[right], s[left]
      left, right = left+1, right-1
