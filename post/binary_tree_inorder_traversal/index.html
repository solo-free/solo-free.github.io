Description：

Given a binary tree, return the inorder traversal of its nodes’ values.

Example:

Input: [1,null,2,3]

Follow up: Recursive solution is trivial, could you do it iteratively?

Thinking:

中序遍历树，即 DFS InOrder, 遵循 left->root->right 的顺序对树中的每个节点进行遍历

    def inorderTraversal(self, root):
        """
        :type root: TreeNode
        :rtype: List[int]
        """
        if root is None:
          return []
        data = []
        if root:
          if root.left:
            data += self.inorderTraversal(root.left)
          data.append(root.val)
          if root.right:
            data += self.inorderTraversal(root.right)
        return data
  • iteration:

    def inorderTraversal(self, root):
        """
        :type root: TreeNode
        :rtype: List[int]
        """
        if root is None:
            return []

        stack = []
        output = []
        cur = root

        while cur or stack:
            while cur is not None:
                stack.append(cur)
                cur = cur.left
            cur = stack.pop()
            output.append(cur.val)
            cur = cur.right
        return output
